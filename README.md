# Getir Test Automation

Desktop and mobile test automation project for Getir using Java, Maven, Selenium, Appium, JUnit 5, Cucumber.

## Reporting

After running the tests, you can find the `test-output` folder under the project folder. You can find the HTML report in the `test-output` folder.
And you can find the screenshots of the failed tests in the `screenshots` folder.

## Test Cases

You can find the test cases in the `src/test/resources/features` folder.
